# Smart Waste Management System (ML-Based)

## Overview
This project implements a data-driven Smart Waste Management System using Machine Learning. The system simulates IoT bin data and predicts future bin fill levels using XGBoost. It also integrates anomaly detection using Isolation Forest to identify unusual waste accumulation patterns.

## Features
- Simulated IoT smart bin dataset
- Exploratory Data Analysis (EDA)
- Predictive modeling using XGBoost Regressor
- Anomaly detection using Isolation Forest
- Urgency classification for proactive waste collection

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib & Seaborn
- Scikit-learn
- XGBoost

## Future Improvements
- Real-time IoT sensor integration
- Route optimization using Vehicle Routing Problem (VRP)
- Deployment using Streamlit dashboard
- Cloud integration

## How to Run
Install dependencies:
pip install -r requirements.txt

Run the model:
python smart_waste_model.py
